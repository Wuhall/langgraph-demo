# LangGraph Demo

LangGraph Tutorial [LangGraph](https://langchain-ai.github.io/langgraph/)  
基于 LangGraph 构建的智能对话系统演示项目。

## 特性

- 🔧 支持工具调用和条件路由
- 🔄 状态管理和消息流处理
- 📚 包含完整的教程示例

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 设置 API Key
cp .env.examples .env

# 运行示例
python tutorial/1.add-tool.ipynb
```
